# Feature 3: Ranking Algorithm Engine

Ranking Algorithm Engine -- Implement the 5 mathematical rating models: Colley Matrix (construct and solve the linear system) and 4 Elo variants (starting ratings 2200, 2400, 2500, 2700). Each model produces an independent rating and rank per team. Compute the unified AggRating (0-100 normalized scale) and AggRank from the 5 model outputs.
